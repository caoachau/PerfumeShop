## 📖 Giới thiệu

**Parfum Shop** là nền tảng thương mại điện tử B2C chuyên biệt cho cửa hàng nước hoa cá nhân/nhỏ lẻ tại Việt Nam. Hệ thống bao gồm cả **cửa hàng dành cho khách hàng** và **admin dashboard quản lý nội bộ**, được thiết kế theo phong cách *Luxury Minimalist* lấy cảm hứng từ các thương hiệu cao cấp như Chanel, Dior, Jo Malone.

```
Mục tiêu: Bán hàng online + Quảng cáo thương hiệu + Quản lý nội bộ
Phạm vi : Website responsive (mobile-first) + Admin dashboard
Ngôn ngữ: Tiếng Việt (giao diện chính), tên sản phẩm hỗ trợ tiếng Anh
```

## 🛠️ Tech Stack

### Core — MERN Stack

| Layer | Công nghệ | Ghi chú |
|-------|-----------|---------|
| **M** — Database | MongoDB Atlas (free 512MB) | NoSQL, linh hoạt schema sản phẩm |
| **E** — Backend | Express.js + Node.js 20 LTS | REST API, middleware auth, business logic |
| **R** — Frontend | React 18 + Vite | SPA, component-based, Vite nhanh hơn CRA |
| **N** — Runtime | Node.js 20 LTS | Runtime chung cho backend & tooling |

### Thư viện & Công cụ

```
Frontend                     Backend                      DevOps / Hosting
──────────────────────────   ──────────────────────────   ──────────────────────
React Router v6              Mongoose (ODM)               Vercel (FE, free)
Zustand (state mgmt)         JWT + bcrypt (auth)          Render.com (BE, free)
Tailwind CSS (styling)       Multer + Cloudinary          MongoDB Atlas M0
Axios (HTTP client)          Resend (email)               Cloudinary (25GB free)
Chart.js (admin charts)      GHN / GHTK API              Google Analytics 4
```

## 🚦 Luồng đặt hàng

```
Khách hàng                                    Hệ thống
──────────                                    ─────────
Xem sản phẩm ──────────────────────────────► GET /api/products
Thêm vào giỏ ──────────────────────────────► POST /api/cart
                                              └─► Kiểm tra tồn kho
Checkout ──────────────────────────────────► POST /api/orders
                                              ├─► Tính phí ship (GHN API)
                                              └─► Tạo Payment record
Thanh toán QR / COD ───────────────────────► PATCH /api/payments/:id
                                              └─► Gửi email xác nhận (Resend)
Theo dõi đơn ──────────────────────────────► GET /api/orders/:id
                                              └─► Sync tracking từ GHN
```

## 🚀 Hướng dẫn cài đặt

### Yêu cầu

- Node.js ≥ 20 LTS
- npm ≥ 10
- Tài khoản: MongoDB Atlas · Cloudinary · Render · Vercel · Resend

### 1. Clone & cài đặt

```bash
git clone https://github.com/YOUR_USERNAME/parfum-shop.git
cd parfum-shop

# Cài backend
cd server && npm install

# Cài frontend
cd ../client && npm install
```

### 2. Cấu hình biến môi trường

```bash
# server/.env
MONGO_URI=mongodb+srv://<user>:<pass>@cluster0.xxxxx.mongodb.net/parfum-shop
JWT_SECRET=your_super_secret_key
JWT_REFRESH_SECRET=your_refresh_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
RESEND_API_KEY=re_xxxxxxxxxxxx
GHN_TOKEN=your_ghn_token
GHN_SHOP_ID=your_shop_id
CLIENT_URL=http://localhost:5173
PORT=5000
```

```bash
# client/.env
VITE_API_URL=http://localhost:5000/api
VITE_GA_MEASUREMENT_ID=G-XXXXXXXXXX
```

### 3. Chạy local

```bash
# Terminal 1 — Backend
cd server && npm run dev

# Terminal 2 — Frontend
cd client && npm run dev
```

Truy cập: `http://localhost:5173` · Admin: `http://localhost:5173/admin`
